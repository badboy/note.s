About note.s

note.s is a simple notes app that can sync all your notes with Dropbox.

To correctly work with Dropbox, a note title can't contain the following characters: 

/ \ | > < : ? * " ;

note.s will silently ignore them, so make sure to set a correct title.

For now all note.s are saved in the “notes” folder in your dropbox. This will be configurable as soon as the dropbox sync is implemented and stable.
